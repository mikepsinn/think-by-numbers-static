## Style Guide Summary

**DO:**
- Be ruthlessly concise
- Use dark humor about death and human stupidity
- Use simple, conversational language
- Create absurd but accurate analogies
- Present solutions as obvious common sense
- Write instructional, actionable content

**DON'T:**
- Use corporate buzzwords or academic jargon
- Try to sound important
- Be earnest or evangelical
- Use euphemisms
- Use overused cliches
- Write like you're selling something
